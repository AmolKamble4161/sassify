Supercharge Your Styles with SASS

SASS (Syntactically Awesome Stylesheets) is a powerful CSS preprocessor that makes your stylesheets clean, reusable, and easier to manage.

SASS Variables

Variables in SASS allow you to store values like colors, fonts, sizes, and reuse them throughout your stylesheets. This makes your code more manageable and consistent.

  • Improve maintainability of styles
  • Change one value, update everywhere
  • Make your design system more scalable

  $primary-color: #6C63FF;

  body {
    background-color: $primary-color;
  }

SASS Mixins

Mixins in SASS let you reuse entire blocks of styles with optional parameters. This makes your code cleaner and more maintainable.

@mixin Definition


  @mixin flex-center {
    display: flex;
    justify-content: center;
    align-items: center;
  }

Usage


  .hero {
    @include flex-center;
  }
  • Reuse groups of CSS rules
  • Add dynamic parameters
  • Keep your code DRY and modular

SASS Functions

Functions in SASS allow you to perform calculations and return values. You can use built-in functions like darken(), or write your own custom ones.

Built-in Function

Using darken() to make a color darker


$base-color: #6C63FF;

.button {
  background-color: darken($base-color, 10%);
}

Custom Function

Create your own spacing scale function


@function calculate-spacing($multiplier) {
  @return $multiplier * 8px;
}

.section {
  padding: calculate-spacing(4);
}
  • Use dynamic logic in your styles
  • Make spacing, colors & sizes consistent
  • Keep code clean and maintainable

SASS @extend

The @extend directive in SASS lets one selector inherit the styles of another, reducing repetition and improving maintainability.

Basic Usage

Reusing styles with @extend


.message {
  padding: 1rem;
  border: 1px solid #ccc;
}

.success {
  @extend .message;
  border-color: green;
}

Placeholder Selectors

Use %placeholders with @extend for cleaner code


%button-base {
  padding: 0.5rem 1rem;
  border-radius: 4px;
}

.btn-primary {
  @extend %button-base;
  background-color: $primary-color;
}
  • Avoid code duplication
  • Use placeholder selectors for clean, DRY styles
  • Easier to maintain shared styling

SASS Operators

Operators in SASS allow you to perform calculations, compare values, and create logic-based styles efficiently.

Arithmetic Operators

Perform calculations like addition, subtraction, multiplication, etc.

$total: 100px + 20px;

Comparison Operators

Compare values using ==, !=, >, <.

@if $width > 500px {
  // styles
}

Logical Operators

Build conditional logic using and, or, not.

@if $light and not $dark {
  // styles
}

String Operators

Concatenate strings using +.

$font-path: "fonts/" + "Roboto";

Color Operations

Modify colors with color-based calculations.

$new-color: #333 + #111;
  • Helps with dynamic spacing and layout adjustments
  • Supports responsive and theme-based design
  • Reduces the need for repetitive hardcoded values
